## Objective

重构路由 data 加载机制，将缓存功能从 renderEach 迁移到 DataLoader，实现统一的缓存管理和插值处理。

**目的：** 提高数据加载的可维护性和性能，通过 DataLoader 集中管理缓存逻辑，避免数据加载分散在多个地方。

**输出：** DataLoader 实现完整的缓存功能，支持插值变量和过期时间；renderEach 简化为只处理钩子逻辑。

## Tasks

### Task 1: DataLoader 实现缓存功能

**Files:** `src/features/data.ts`

**Action:**
1. 在 DataLoader 类中添加私有属性 `cached: Map<string, any>` 用于缓存数据
2. 添加缓存项接口定义：
   ```typescript
   interface CacheItem {
       data: Record<string, any>;
       expires: number; // 0 表示不过期
       timestamp: number;
   }
   ```
3. 在 `loadData` 方法中实现缓存逻辑：
   - 根据 RouteDataOptions.cache 参数决定是否启用缓存
   - 当 cache=true 时，默认缓存 Key 为 "{path}"
   - 使用 src/utils/params.ts 的 params 函数处理缓存 Key 插值
   - 支持的插值变量：path、basepath、url、timestamp、query、params
   - 检查缓存是否过期（expires 参数，默认 0 代表不过期）
   - 如果缓存有效，直接返回缓存数据
   - 如果缓存无效或不存在，加载数据后存入缓存
4. 实现 `clearCache(key?: string)` 方法：
   - 不传参数时清空所有缓存
   - 传参数时清空指定缓存

**Implementation Details:**
- 插值处理：使用 `params.params(cacheKey, { path, basepath, url, timestamp, ...query, ...params })`
- 过期检查：`expires === 0 || Date.now() - timestamp < expires`
- 从 router 实例获取当前路由信息：`this._router.routes.current`

**Verification:**
```typescript

const dataLoader = new DataLoader(router);
const route = {
    path: '/users/:id',
    data: {
        from: async () => ({ user: 'test' }),
        cache: true, // 默认缓存 Key 为 "{path}"
        expires: 5000 // 5秒过期
    }
};

// 第一次加载，应该从远程加载
const result1 = await dataLoader.loadData(route);
// 第二次加载，应该从缓存加载
const result2 = await dataLoader.loadData(route);
// 验证缓存生效
```

**Done:**
- DataLoader 实现了完整的缓存功能
- 缓存 Key 支持插值变量
- 支持过期时间检查
- 提供清理缓存的方法

   - query: 当前路由的查询参数
   - params: 当前路由的路径参数
3. 修改 `loadRemoteData` 方法，支持插值后的 URL 加载

**Implementation Details:**

```typescript
// 构建插值字典

    ...this._router.routes.current.query,
    ...this._router.routes.current.params
```

**Files:** `src/features/hooks.ts`

**Action:**
1. 修改 `executeRenderEach` 方法：
   - 移除数据合并逻辑

   - 返回类型改为 `Promise<void>`

   - 只调用钩子函数

**Implementation Details:**
```typescript
// 修改后的 executeRenderEach 方法
async executeRenderEach(to: RouteItem, from: RouteItem): Promise<void> {
    const globalHooks = this.hooks[HookTypeValues.RENDER_EACH as HookType] as RenderEachHook[];
    const routeHooks = to.renderEach
        ? Array.isArray(to.renderEach)
            ? to.renderEach
            : [to.renderEach]
        : [];

    const allHooks = [...globalHooks, ...routeHooks];
    if (allHooks.length === 0) return;

    for (const hook of allHooks) {
        try {
            await this.runRenderEachHook(hook, to, from);
        } catch (error) {
            console.error("[Router] renderEach hook failed:", {
                route: to.name,
                error,
            });
        }
    }
}
```

