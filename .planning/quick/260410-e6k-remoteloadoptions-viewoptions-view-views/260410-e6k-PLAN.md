## 实现细节

### 类型守卫函数

需要添加一个类型守卫函数来区分 `ViewSource` 和 `ViewOptions`：

```typescript
/**
 * 判断 view 配置是否为 ViewOptions
 */
function isViewOptions(view: ViewSource | ViewOptions): view is ViewOptions {
    return typeof view === 'object' && view !== null && 'form' in view;
}
```

### 默认值处理

当 `view` 是 `ViewSource` 时，使用以下默认值：
- `allowUnsafeHTML`: false
- `timeout`: 5000
- `selector`: undefined
