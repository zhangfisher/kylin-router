**文件：** [src/types/events.ts](src/types/events.ts)

**操作：**
1. 将所有现有事件名称的 `-` 分隔符改为 `/`：
   - `navigation-start` → `navigation/start`
   - `navigation-end` → `navigation/end`
   - `route-change` → `route/change`

   - `modal-close` → `modal/close`
2. 添加新事件类型定义：

   - `view/loaded`: 视图模板加载完成后触发，参数 `{ route: RouteItem }`

**验证：**
- TypeScript 编译无错误
